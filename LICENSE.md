## FAQ

### Can I use these recipes commercially?
**Yes**, with attribution. If you sell books, run a restaurant, or create products:
- Use recipes freely
- Create original content (photos, descriptions)
- Credit The Science Cookbook
- If you publish adaptations, use CC BY-SA 4.0

### Can I keep my adaptations secret?
**No**, CC BY-SA requires you to license adaptations under the same license. If you want to keep something proprietary, don't use these recipes as the base.

### What if I improve a recipe?
**Please share it back!** 
- Post in comments
- Create a pull request
- Write a blog post (with attribution)
- Email improvements

Improvements benefit everyone.

### Can I use these for a book?
**Yes!** Books can be published under CC BY-SA 4.0. Your book would be:
- Free to share (must include full license)
- Adaptable by others (ShareAlike)
- Profitable for you (commercial use is allowed)

If you want more control, write original recipes instead.

### Does this cover photos?
**No.** My photos remain under copyright. You must:
- Create your own photos
- Get permission to use mine
- Credit if you link to them

The *content* of recipes is CC BY-SA, not the *images*.
